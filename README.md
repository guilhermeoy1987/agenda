# Projeto Agenda de Contatos - EBAC

Este repositório contém o desenvolvimento de uma agenda interativa para cadastro de nomes e telefones, com foco em manipulação de tabelas e validação de dados.

## O que foi realizado:
* **Cadastro Dinâmico:** Implementação de lógica para adicionar contatos à lista sem recarregar a página.
* **Validação de Entradas:** Criação de filtros para evitar nomes duplicados e garantir o preenchimento correto dos campos.
* **Interface Organizada:** Uso de tabelas HTML5 e CSS para exibição clara das informações cadastradas.
* **Formatação de Dados:** Aplicação de máscaras para campos de telefone, melhorando a visualização.

## Tecnologias:
* HTML5 / CSS3
* JavaScript / jQuery
* Git / GitHub
